- a)
Estratégia gananciosa poderia passar por colocar um cartaz e colocar o próximo cartaz a mais de 5km
```cpp
int pos[n];
int valor[n];
int total = valor[0];
int anterior = pos[0];

for (int i = 1; i < n; i++){
    if (anterior + 5 >= pos[i]){
        total += pos;
        anterior = pos[i];
    }
}
```

Complexidade O(n).
O algoritmo não é ótimo pois apenas tem em conta as distâncias entre cada posição e não os valores a cobrar.

            if custo(w) > custo(v) + 10*preco(w)
                gasolina += 10
                custo(w) = custo(v) + 10*preco(w)
                gasolina -= weight(v, w)
                path(w) = v

                    insert(Q, w)
                else
                    decrease-key(Q, w)
        else
            if custo(w) > custo(v)
                custo(w) = custo(v)
                gasolina -= weight(v, w)
                path(w) = v
                if w ∉ Q
                    insert(Q, w)
                else
                    decrease-key(Q, w)
